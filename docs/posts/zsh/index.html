Instalar y configurar ZSH.

[BORRADOR]

1.- Descarga oh-my-zsh: - sudo sh -c “$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

2.- Agregar theme: - En .oh-my-zsh/themes/amuse_2021.zsh-theme

3.- Seleccionar el theme: - Con nano .zshrc agregamos el themea “amuse-2021”

Cambiar de shell: -$ chsh -s /bin/zsh
